###GET STARTED

Jeckyll is a ruby gem, you need to install Ruby and then install Jeckyll.

```
$ruby --version
	ruby 2.0.0p645 (2015-04-13 revision 50299) 

$sudo gem install jekyll
```

###SETUP
```
$cd GIT_PROJECT_HOME
$bundle install
```
###START LOCALHOST:4000 SERVICE
```
$bundle exec jekyll serve
```
